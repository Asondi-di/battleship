# Подробная инструкция по Battleship LAN

## 1) Что это

`Battleship LAN` — это локальная многопользовательская версия «Морского боя»:
- сервер на Node.js хранит комнаты и состояние партии;
- игроки подключаются через Chrome Extension по WebSocket;
- в одной комнате играют **2 игрока**.

## 2) Подготовка сервера (хост игры)

1. Откройте терминал в папке проекта.
2. Установите зависимости:

```bash
npm install
```

3. Запустите сервер:

```bash
npm start
```

4. Узнайте IP вашего компьютера в LAN (например, `192.168.1.20`).
5. Убедитесь, что порт `3000` открыт в локальном фаерволе.

## 3) Установка расширения в Chrome (каждому игроку)

1. Откройте `chrome://extensions`.
2. Включите **Режим разработчика**.
3. Нажмите **Загрузить распакованное расширение**.
4. Выберите папку `extension` из этого проекта.
5. Закрепите расширение на панели Chrome (по желанию).

## 4) Как подключиться к игре

1. Нажмите иконку расширения, чтобы открыть popup.
2. В поле **Сервер WebSocket** введите адрес сервера:
    - хост: `ws://192.168.1.20:3000`
    - второй игрок (или тот же ПК для теста): тот же адрес.
3. В поле **Комната** введите одно и то же имя комнаты у обоих игроков.
4. Нажмите **Подключиться**.

## 5) Как начать матч

1. Нажмите **Авторасстановка кораблей**.
2. Проверьте своё поле (корабли подсвечены).
3. Нажмите **Подтвердить расстановку**.
4. Когда оба игрока готовы, сервер автоматически запускает партию.

## 6) Как играть (для обоих игроков)

1. Статус «Ваш ход» означает, что можно стрелять по полю соперника.
2. Клик по клетке на поле противника отправляет выстрел.
3. Логика ходов:
    - попадание (`красный`) — ход остаётся у атакующего;
    - промах (`синий`) — ход переходит сопернику.
4. Побеждает игрок, потопивший все корабли противника.

## 7) Обозначения на полях

### Ваше поле
- **фиолетовый** — ваш корабль;
- **красный** — по вашему кораблю попали;
- **синий** — выстрел соперника в воду.

### Поле соперника
- **красный** — ваше попадание;
- **синий** — ваш промах.

## 8) Типичные проблемы и решения

### Не подключается к серверу
- Проверьте правильность URL (`ws://IP:3000`).
- Проверьте, что сервер запущен.
- Проверьте, что игроки в одной сети.
- Проверьте фаервол/антивирус (доступ к порту 3000).

### Второй игрок не может войти в комнату
- В комнате уже 2 игрока.
- Создайте новую комнату (другое название).

### Игра зависла после выхода игрока
- Это нормальное поведение текущей версии.
- Оставшийся игрок должен заново нажать «Авторасстановка» и «Подтвердить расстановку», когда второй снова подключится.

## 9) Быстрый сценарий для организатора LAN-игры

1. Поднять сервер на своём ПК.
2. Сказать всем участникам свой LAN IP.
3. Убедиться, что у всех загружено расширение.
4. Назначить пары и имена комнат (например, `table1`, `table2`).
5. Старт: расстановка → готовность → игра.
